# HyBrAIn — The Human × AI Interface for Sustainable Use

An interaction regulation system that detects cognitive load, tracks trust, and intervenes to make AI usage consistent and sustainable.

## Quick Start

```bash
npm install
npm run dev
```

Opens at `http://localhost:3000`

## Build

```bash
npm run build
```

Output in `dist/`

## Stack

- React 18
- Vite 5
- Recharts
- Zero external CSS — all inline styles

## Deploy to Vercel

1. Push this repo to GitHub
2. Go to [vercel.com/new](https://vercel.com/new)
3. Import the repo
4. Deploy (Vercel auto-detects Vite)

---

*HyBrAIn is a product interaction layer for sustainable AI usage, not a medical or clinical system.*
